Pill

The Pill component is great for representing simple items in the UI: assets, tags, categories, teams, or users.

Pills can have an icon and an optional "Delete" button which should remove the associated item from the list it's in.

Pills can be made clickable, but should never be used as action buttons (think "Confirm" or "Cancel" buttons). Use the Button component for this use case.

Interactive examples

To make a Pill's body clickable, simply attach an onClick handler. You can also make the Pill a link: Use the as prop to choose which element to render, e.g. "a" or react-router's {Link}. You can then use the href or to prop to define the link's destination. All other link props will also be passed through to the clickable element.

Props

In addition to the props below, the Pill component also supports the margin style props (e.g. m, ml, mx etc).

icon
string

Name of the icon to be shown

iconColor
string

Set the color of the icon. Can be a primitive colour value or the name of a colorBlock. If theres a colorBlock of the same name as the chosen icon, it will automatically be used.

iconRenderer
func
function defaultIconRenderer({iconName, iconColor}) { if (!iconName) { return null; } return ( <ThemeSection name="colorBlock" colorBlock={iconColor || iconName}> <IconWrapper> <Icon name={iconName} data-iconname={iconName} /> </IconWrapper> </ThemeSection> ); }

Customise the way the icons are rendered. Will be called with an object containing the properties iconName and iconColor.

background
string

Set the background color of the pill.

dimmed
bool

Dims the whole component slightly. Use sparingly and ONLY to indicate e.g. partial matches in a group of items.

id
string

When an ID is passed, it will be applied to the main pill element with the suffix _wrapper, and to the delete button with the suffix _side_button

onDelete
func

Callback thats run when the delete button is activated. Hides the delete button, if not provided. Remember to provide a deleteLabel prop.

deleteLabel
string

Accessible label for the delete button.
