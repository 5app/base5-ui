# ProgressCircle

```jsx
import ProgressCircle from 'base5-ui/ProgressCircle';
```

A component for visually representing the progress of something, a numerical value within a range, or other data that can be displayed as a fraction of a circle.

By default, this component renders an element with the role 'progress bar'. Please read the WAI-ARIA guidelines about the [meter](https://www.w3.org/TR/wai-aria-1.2/#meter) and [progressbar](https://www.w3.org/TR/wai-aria-1.2/#progressbar) roles to determine which role to use. For use cases that don't map to one of these roles, set it to 'none' and provide meaning via the `a11yLabel` prop.
