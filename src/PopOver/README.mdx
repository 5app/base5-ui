# PopOver

PopOver is a low-level building block used to attach an element (the 'popover') to another element (the 'reference element').

It's a wrapper around [react-popper](https://github.com/FezVrasta/react-popper) and useful for building UI patterns like tooltips, dropdown menus, and floating hints.

<Playground>
	<Box p="l">
		<PopOver
			isOpen
			content="I'm a popover. I'll always stick to my reference element!"
			placement="top-start"
		>
			{popover => <span ref={popover.ref}>I'm a reference element</span>}
		</PopOver>
	</Box>
</Playground>
