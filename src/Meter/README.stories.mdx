# Meter

An accessible stand-in for the native `<meter>` HTML element built to ensure consistent visuals across browsers.

If you don't define a color for the meter, it will look for color blocks named 'negative', 'neutral', and 'positive' in your global theme, and use them for the sections of the meter that were defined using the `low` and `high` props.

You can also visually prevent the meter value from falling below a certain value by specifying the `visualMin` prop.
