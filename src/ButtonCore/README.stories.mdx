# ButtonCore

```js
import ButtonCore from 'base5-ui/ButtonCore';
```

Base component for building interactive elements like buttons and links.

Its styling is neutral: By default it just blends in with its surrounding text. Make sure to add sufficiently contrasting visual cues about its interactivity to the component when using it.
