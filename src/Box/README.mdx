# Box

`Box` is a generic layout primitive, exposing many useful props for controlling style and layout of the component.

The following properties can be changed:

-   Position
-   Size
-   Margin and padding
-   Flex behaviour (grow, shrink, basis, flexAlign)
-   Borders & box-shadow
-   Background & text colours
-   Text styles (bold, dimmed, textAlign, overflow, caps)

## Examples

<Playground>
	<Box borderBottom="thin" pb="xxs" pl="xs">
		Hello
	</Box>
	<Box bold breakpoints="s" mt={['xs', 'l']} p="s" boxShadow="soft">
		Hello
	</Box>
	<Box mt={['xs', 'l']} p="s" color="text" background="shaded">
		Hello
	</Box>
	<Box
		mt={['xs', 'l']}
		p="s"
		background={theme =>
			mix(getSolidBackgroundShade(theme), 'rgb')(theme.links, 0.75)
		}
	>
		Hello
	</Box>
</Playground>
