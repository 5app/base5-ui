# Select

```jsx
import {Select, Option} from 'base5-ui/Select';
```

A custom select menu component that allows users to select one option from a list of predefined options.

This component largely follows the implementation recommendations from the [WAI-ARIA Authoring Practices 1.2 working draft](https://www.w3.org/TR/wai-aria-practices-1.2/#combobox), with adjustments made based on Sarah Higley's article [Select Your Poison Part 2](https://www.24a11y.com/2019/select-your-poison-part-2/).

There are known issues around this pattern when using VoiceOver with Safari on macOS.
