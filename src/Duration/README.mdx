# Duration

A simple component for rendering a duration (e.g. of a video) with an accessible label. Takes a `value` prop in seconds.

By default, the duration is rendered "YouTube-style" in the format `m:s` or `h:m:s`, but you can pass a render function to the children prop to easily create a custom readout.

## Examples

<Playground>
	<Duration value={null} />
	<br />
	<Duration value={2.7245} />
	<br />
	<Duration value={27} />
	<br />
	<Duration value="489" />
	<br />
	<Duration value={1104} />
	<br />
	<Duration value={7729} />
	<br />
	<Duration value={7729}>
		{dur => `${dur.hours}h ${dur.minutes}m ${dur.seconds}s`}
	</Duration>
	<br />
	<Duration value={7729}>{dur => `${dur.fullHours} hours`}</Duration>
	<br />
	<Duration value={7729}>{dur => `${dur.fullMinutes} mins`}</Duration>
</Playground>
