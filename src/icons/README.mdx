# Icon Overview

5app's SVG icons as React components, powered by styled-components.

## Adding new icons

1. Add new SVG icons to [src-icons/svg](/src-icons/svg)
2. Add any new icons as an export inside of [src/icons/index.js](src/icons/index.js)
3. To make new icons available from the `<Icon />` component, define a `string` name for them in the icon map in [src/Icon/iconMap.js](src/Icon/iconMap.js)
4. Add new icons to this README for documentation

Once that's done, run `npm run build-icons` to convert the SVGs to standalone React components inside of [src/icons](/src/icons).

After that, you can follow the normal instructions for updating/publishing this repo found in the base5-UI Introduction.
