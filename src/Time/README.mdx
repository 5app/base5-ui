# Time

`Time` is a helper component for providing time given a context. For example if something happened a few seconds ago, let's say it happened a "seconds ago", similarly if it happened today, provide the time "3pm" rather than the full date, and dont provide too much information, seconds aren't relevant when we're talking hours.

On hover over the full time is provided.

## Props

<Props of={Time} />

- `dateTime`: Any Date string recognized by [Date.parse()](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date/parse). Note: Suffix with `Z` to define the date as a UTC value or offset.
