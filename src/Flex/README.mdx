# Flex

`Flex` is a container component for building layouts using CSS flexbox. It's especially useful when used with the `Box` component which provides control for laying out the children.

The component is built on top of `Box` and its props `align`, `column`, and `wrap` are aliases for the `Box` props `alignItems`, `stackItems`, and `wrapItems`.
