# MenuList

```jsx
import {Wrapper, Item, ItemIcon, Link, ItemDivider, GroupLabel} from 'base5-ui/MenuList';
// or
import * as MenuList from 'base5-ui/MenuList';
```

MenuList is a set of components used for building dropdown menus:

-   MenuList.Wrapper
-   MenuList.Item
-   MenuList.ItemIcon
-   MenuList.Link
-   MenuList.ItemDivider
-   MenuList.GroupLabel

They don't come with any built-in behaviour. Check out the <LinkTo kind="Components/Menu">Menu</LinkTo> and <LinkTo kind="Components/Menu (Nav)">NavMenu</LinkTo> components to see the MenuList component in use.

							<>
								<MenuList.Item key={item.id}>

										isHighlighted={item.id === highlightedItem}
										isDisabled={item.isDisabled}

										}

										{item.label}

								{index === 2 && <MenuList.ItemDivider/> }
							</>
						))}
					</MenuList.Wrapper>
				</PopoverCard>
			);
		}}
