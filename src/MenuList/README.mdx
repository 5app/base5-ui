# MenuList

MenuList is a set of components used for building dropdown menus:

-   MenuList.Wrapper
-   MenuList.Item
-   MenuList.ItemIcon
-   MenuList.Link

They don't come with any built-in behaviour. See the [Menu](src-menu-readme) and [NavMenu](src-nav-menu-readme) components to see the MenuList component in use.

<Playground>
	{() => {
		const [highlightedItem, setHighlightedItem] = React.useState(0);
		const [selectedItem, setSelectedItem] = React.useState(0);
		const menuItems = [
			{id: 0, label: 'Insert', icon: 'plus'},
			{id: 1, label: 'Favourite', icon: 'star'},
			{id: 2, label: 'Insights', icon: 'trending', isDisabled: true},
			{id: 3, label: 'Download', icon: 'download'},
		];
		return (
			<PopoverCard isOpen renderInPlace>
				<MenuList.Wrapper>
					{menuItems.map((item, index) => (
						<MenuList.Item key={item.id}>
							<MenuList.Link
								isActive={item.id === selectedItem}
								isHighlighted={item.id === highlightedItem}
								isDisabled={item.isDisabled}
								onMouseEnter={() => setHighlightedItem(item.id)}
							>
								<MenuList.ItemIcon name={item.icon} />
								{item.label}
							</MenuList.Link>
						</MenuList.Item>
					))}
				</MenuList.Wrapper>
			</PopoverCard>
		);
	}}
</Playground>
