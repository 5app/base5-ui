# Tooltip

Add a tooltip or dropdown menu to other interactive elements. This component is a combination of the `Popover` component and the `usePopoverState` hook which provides the open/close state and functionality.

## Example

The below is a simplified implementation of the [TooltipButton](/src-tooltip-button-readme) component.

<Playground>
	<Box p="l">
		<Tooltip
			renderWhenClosed
			delay={250}
			placement="top"
			content="Hi, I am a useful tooltip"
			id="tooltip-id"
		>
			{popover => (
				<Button
					ref={popover.ref}
					aria-describedby="tooltip-id"
					onMouseEnter={popover.open}
					onMouseLeave={popover.close}
					onFocus={popover.open}
					onBlur={popover.close}
					onMouseDown={popover.close}
				>
					Show tooltip
				</Button>
			)}
		</Tooltip>
	</Box>
</Playground>
